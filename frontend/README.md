# Frontend Output Directory

This directory stores the user interface prototypes generated by the AI Agents. 

Because our architecture uses **Google Stitch** (stitch.withgoogle.com) as the frontend rapid prototyping tool, all files generated in this directory will be single-file HTML/Tailwind templates. 

When your cloud agent completes a UI task, it will drop the resulting code here. You can then upload it to Google Stitch or Vercel.
